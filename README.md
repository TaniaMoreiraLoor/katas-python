# 🐍 Fundamentos de Python: Estructuras de Datos, Funciones y Programación Orientada a Objetos
  

## 📖 Descripción del Proyecto

  

El objetivo principal es aplicar los conceptos fundamentales del lenguaje Python mediante la resolución de una colección de ejercicios prácticos de lógica y programación. A lo largo del proyecto se desarrollan soluciones para distintos problemas utilizando estructuras de datos, funciones, programación orientada a objetos, manejo de excepciones y funciones incorporadas de Python.

  

## 🎯 Objetivos

  

- Aplicar los fundamentos de programación en Python.

- Trabajar con diferentes estructuras de datos.

- Utilizar funciones y programación modular.

- Implementar estructuras condicionales e iterativas.

- Gestionar errores mediante excepciones.

- Utilizar funciones `lambda`, `map()`, `filter()` y `reduce()`.

- Comprender los principios básicos de la programación orientada a objetos.

- Aplicar buenas prácticas de programación.
